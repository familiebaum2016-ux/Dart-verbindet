# Dart verbindet - Mobile E-Dart Hilfe

Eine hochprofessionelle Website für mobile E-Dart Hilfe mit Fokus auf Inklusion, Gemeinschaft und Spaß.

## 📋 Inhalte

- **Hero-Section**: Ansprechende Einleitung mit "Mobile E-Dart Hilfe" und Motto "Spaß. Teilhabe. Gemeinschaft."
- **Über mich**: Vorstellung von Tim Baum, Gründer von Dart verbindet
- **Mein Angebot**: Vier Service-Karten mit den Hauptleistungen
- **Meine Aufgaben**: Detaillierte Auflistung der Verantwortungen
- **Zielgruppen**: Übersicht der Einrichtungen (Wohnheime, Werkstätten, Tagesförder-stätten, Förder-schulen)
- **Kontakt**: Telefon, Website, E-Mail und Kontaktformular
- **Footer**: Impressum und Datenschutz Links

## 🎨 Design

- **Farben**: Grün (#16a34a), Blau (#2563eb), Rot (#dc2626) - basierend auf dem Flyer
- **Typografie**: Moderne, lesbare Fonts mit klarer Hierarchie
- **Layout**: Responsive Design für Desktop, Tablet und Mobile
- **Interaktivität**: Hover-Effekte, Scroll-Animationen und benutzerfreundliche Navigation

## 🚀 Verwendung

1. **Lokal öffnen**: Einfach die `index.html` im Browser öffnen
2. **Auf GitHub Pages hosten**:
   - Repository erstellen
   - Diese Dateien hochladen
   - In den Repository-Einstellungen GitHub Pages aktivieren
   - Website ist dann unter `https://[dein-username].github.io/[repo-name]` erreichbar

## 📝 Anpassungen

### Kontaktformular aktivieren
Das Kontaktformular ist derzeit statisch. Um es funktionsfähig zu machen:
- Verwende einen Service wie [Formspree](https://formspree.io/) oder [Basin](https://usebasin.com/)
- Oder integriere ein Backend-System

### Bilder hinzufügen
- Ersetze die Emojis mit echten Fotos von Tim Baum
- Speichere Bilder im `assets/` Ordner
- Passe die Bild-Pfade in der HTML an

### Logo einbinden
- Platziere das "Dart verbindet" Logo im `assets/` Ordner
- Aktualisiere die Navigation und den Favicon
